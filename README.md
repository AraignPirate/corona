<h1 align="center">CORONA</h1>

## About


A command line bash script to get *statistics corona virus update*.

- Required **BASH**

BUILD ON : Kali Linux.

## Usage

Clone the repo using **bash terminal** and run these commands

> `git clone https://github.com/AraignPirate/corona.git`

> `cd corona`

> `chmod +x corona.sh`

> `./corona.sh`
